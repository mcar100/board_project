# Board Project
클라이언트 단 React로 구현
서버 단 SpringBoot로 구현

- 로그인, 회원가입 (+유효성 검사, 이메일 인증)
- 게시글 CRUD (+파일 업로드)
- 댓글, 대댓글 CRUD
